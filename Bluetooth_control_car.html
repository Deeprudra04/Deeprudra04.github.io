This project demonstrates a remote-controlled robotic car powered by an Arduino UNO and HC-05 Bluetooth module, enabling wireless operation via a mobile app. The system uses 300 RPM gear motors, an L298N motor driver, and a custom-coded Arduino program to interpret directional commands sent over Bluetooth. It showcases fundamental concepts in embedded systems, motor control, serial communication, and mobile interfacing, making it an excellent hands-on application in robotics and IoT.
